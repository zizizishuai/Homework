# 第三题：姓名新闻搜索系统

## 概述

本程序是**人工智能概论大作业**的第三题，实现了基于百度新闻的姓名新闻搜索系统。

## 功能特性

### 核心功能
- 🔍 **百度新闻搜索** - 从百度新闻搜索指定姓名的相关新闻
- 📄 **多页搜索** - 支持1-10页的搜索范围设置
- 🏷️ **智能过滤** - 自动过滤不相关新闻（姓名必须出现在标题或摘要中）
- 🗂️ **来源筛选** - 按新闻来源进行筛选
- ↕️ **多种排序** - 按姓名出现次数、时间、来源进行排序
- 📊 **统计分析** - 可视化展示搜索结果的统计信息（4个图表）
- 📜 **历史记录** - 保存搜索历史，方便重复查询
- 📥 **CSV导出** - 将搜索结果导出为CSV文件

### 技术实现
- **网络爬虫**：使用 requests 库发送HTTP请求
- **HTML解析**：使用 BeautifulSoup 解析页面
- **反爬机制**：随机User-Agent、请求间隔、安全验证检测
- **GUI框架**：PyQt5 跨平台图形界面
- **数据可视化**：matplotlib 绘制统计图表
- **多线程**：QThread 实现异步搜索，避免界面阻塞

## 运行方式

### 方式一：作为独立程序运行

```bash
cd news_search
python main.py
```

### 方式二：作为集成系统的一部分运行

从项目根目录运行主程序：

```bash
cd integrated_system
python main.py
```

然后在界面的"姓名新闻搜索"标签页中使用。

## 使用方法

1. 在搜索框中输入要查询的姓名
2. 设置搜索页数（1-10页）
3. 选择搜索范围（全部新闻或指定来源）
4. 点击"开始搜索"按钮
5. 等待搜索完成，查看结果
6. 双击表格行可在浏览器中打开新闻原文
