# 姓名新闻搜索系统

一个基于 Python 和 Tkinter 的姓名新闻搜索系统，通过百度新闻搜索指定姓名的相关新闻，并提供可视化统计分析。

## 🎯 功能介绍

### 新闻搜索
- **姓名搜索**：输入姓名搜索相关新闻
- **多页搜索**：支持翻页获取更多新闻
- **来源筛选**：可按新闻来源网站筛选
- **安全验证处理**：自动处理百度安全验证

### 结果展示
- **新闻列表**：显示标题、来源、时间、摘要
- **详情预览**：点击查看新闻摘要
- **链接跳转**：点击链接打开新闻原文
- **搜索统计**：显示搜索结果数量和耗时

### 数据导出
- **Excel导出**：将搜索结果导出为Excel文件
- **JSON导出**：支持JSON格式导出
- **搜索历史**：自动保存搜索记录

### 统计分析
- **来源分布**：统计各新闻来源数量
- **时间分布**：按日期统计新闻数量
- **可视化图表**：柱状图展示统计结果

## 🛠️ 技术栈

- **语言**: Python 3.x
- **界面**: Tkinter
- **网络请求**: requests
- **HTML解析**: BeautifulSoup4
- **数据处理**: pandas
- **可视化**: matplotlib
- **随机UA**: fake_useragent

## 📁 文件说明

| 文件 | 说明 |
|------|------|
| `name_news_search.py` | 主程序入口，包含GUI界面和新闻搜索逻辑 |
| `config.py` | 配置文件，包含搜索URL、超时时间等设置 |
| `logger.py` | 日志模块，记录运行日志 |
| `search_history.json` | 搜索历史记录（自动生成） |
| `news_search.log` | 运行日志文件（自动生成） |

## 📖 使用说明

### 1. 启动程序

```powershell
python 第三题\name_news_search.py
```

### 2. 搜索新闻

1. 在搜索框中输入要搜索的姓名
2. 点击"搜索"按钮或按回车键
3. 等待搜索结果显示

### 3. 浏览结果

- **查看新闻列表**：显示标题、来源、发布时间、摘要
- **点击标题**：查看新闻摘要详情
- **点击来源链接**：在浏览器中打开原文
- **翻页**：点击"上一页"/"下一页"浏览更多结果

### 4. 筛选功能

- **来源筛选**：在下拉框中选择特定新闻来源
- **关键词高亮**：搜索关键词会在结果中高亮显示

### 5. 数据导出

- **导出Excel**：点击"导出Excel"保存搜索结果
- **导出JSON**：点击"导出JSON"保存数据
- **查看历史**：点击"搜索历史"查看之前的搜索记录

### 6. 统计分析

- 点击"统计分析"标签页
- 查看新闻来源分布柱状图
- 查看新闻时间分布柱状图

## ⚠️ 注意事项

- 本程序仅用于学习和研究目的
- 请遵守百度搜索的使用规则
- 搜索频率不宜过快，避免触发安全验证
- 部分网站可能禁止爬虫，部分新闻链接可能无法访问

## ❓ 常见问题

**Q: 搜索结果为空？**
A: 可能是该姓名在百度新闻中没有相关报道，或触发了安全验证。

**Q: 如何提高搜索成功率？**
A: 程序会自动处理安全验证，若频繁失败请稍后再试。

**Q: 支持哪些新闻来源？**
A: 支持百度新闻索引的所有新闻网站，包括新浪、网易、腾讯等主流媒体。

**Q: 导出的Excel文件在哪里？**
A: 默认保存在程序运行目录下，文件名格式为 `news_search_姓名_日期.xlsx`。
